# Expanse calculator backend

## setup

### install dependencies

```shell
npm install
```

### run development server

```shell
npm run dev
```

## API

### Login
```shell
POST /api/v1/user/login
```

### Register
```shell
POST /api/v1/user/register
```

### Refresh Token
```shell
POST /api/v1/user/refresh-token
```

### List
```shell
GET /api/v1/user/list
```

### Age of user
```shell
GET /api/v1/user/
```
